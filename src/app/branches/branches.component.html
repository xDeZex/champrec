These are the technologies and languages used to make this website:

.NET

This website requests data from a .NET API hosted on AWS.

Angular

This website is coded in Angular.

AWS

This website utilizes both AWS Lambda and AWS DynamoDB.

AWS Lambda

Lambda is used to host the .NET API and a Python script.

AWS DynamoDB

DynamoDB is used to store submitted names and is automatically updated from the Riot API by a Lambda function.

GIT

My projects are on Github and this website is hosted on Github Pages.

Python

Python is used to write smaller scripts and AWS triggers for DynamoDB.
